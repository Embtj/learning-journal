# Learning Journal

This is a solo project built as part of the Scrimba Frontend Developer Career Path. 
The design was based on a Figma mockup and implemented using a mobile-first approach.

## Technologies
- HTML
- CSS

## Features
- Responsive layout
- Mobile-first design
- Featured blog post
- Grid layout for blog posts

## What I learned
This project helped me practice responsive design, organizing CSS, and working from a Figma design.
